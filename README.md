# 🔐 Password Generator (Python)

A Python tool that generates strong random passwords using letters, numbers, and symbols.

---

## 🚀 Features
- Generates secure random passwords
- User defines password length
- Uses letters, numbers, and special characters
- Simple and fast CLI tool

---

## 🛠 Technologies Used
- Python
- random module
- string module

---

## ▶️ How to Run
1. Install Python
2. Open `password_generator.py`
3. Run the program

---

## 📌 Purpose
Built to learn Python randomness and improve understanding of basic cybersecurity concepts.
